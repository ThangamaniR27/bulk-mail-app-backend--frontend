# Bulk Mail App

A full stack Bulk Mail Application built using React, Node.js, Express, MongoDB, and Nodemailer.

## Features

- Send bulk emails
- React frontend UI
- Node.js + Express backend
- MongoDB database connection
- Nodemailer email integration
- REST API support

---

## Tech Stack

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Nodemailer
- Cors
